# 収支管理フォーム

## Overview
HTMLで作成したフォームに入力しPythonで処理をしExcelに反映する  
ページ内にシート作成ボタンを配置し年度始めにクリックする  
フォームに入力した内容（ログ）はあとで見返せるようにしたい  
現在時刻と日付を表示させる  
貯金額も表示させる  
レスポンシブ対応させる（スマホ&PC)  

## Requirement
-Windodws11  
-HTML5  
-CSS3  
-Python3.11  

## Usage
1.フォームを入力  
2.Python(Flask&Pandaas)で処理  
3.excelに反映
