# 収支管理フォーム

## Overview
HTMLで作成したフォームに入力しPythonで処理をしてexcelに反映する

## Requirement
-Windodws11  
-HTML5  
-CSS3  
-Python3.11  

## Usage
1.フォームを入力  
2.Python(Openpyxl)で処理  
3.excelに反映
