# 収支管理フォーム

## Overview
1.HTMLで作成したフォームに入力しPythonで処理をしExcelに反映する  
2.ページ内にシート作成ボタンを配置し年度始めにクリックする  
3.フォームに入力した内容（ログ）はあとで見返せるようにしたい  
4.現在時刻と日付を表示させる  
5.貯金額も表示させる  
6.レスポンシブ対応させる（スマホ&PC)  

## Requirement
-Windodws11  
-HTML5  
-CSS3  
-Python3.11  

## Usage
1.フォームを入力  
2.Python(Flask&Pandaas)で処理  
3.excelに反映
